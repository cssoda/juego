Estás por aprender más acerca de los selectores de CSS! Los Selectores son las sentencias que te permiten seleccionar a cuáles elementos aplicarle los estilos css que defines en una regla de css.

Por ejemplo... en esta regla CSS

p {
   margin-bottom: 12px;
}

Aquí, la "p" es el selector (en este ejemplo: selecciona todos los elementos p que son los párrafos en html) y les aplica ese estilo de margen inferior (margin-bottom).

Para jugar, escribe un selector CSS en el editor de abajo, para seleccionar los elementos correctos de la mesa. Si los seleccionas correctamente, avanzas al siguiente nivel!.

Pasa el puntero del mouse encima los items en la mesa para que veas cuál es su código correspondiente en HTML.

Además, puedes conseguir ayuda importante leyendo las pistas acerca de cada selector, en la caja a la derecha! →
